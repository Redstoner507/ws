# WS

## Description

`ws` est une commande qui affiche des animations **ASCII art** de bateaux dans le terminal.<br>  
Elle est inspirée de la célèbre commande [`sl`](https://github.com/mtoyoda/sl), mais adaptée à un univers naval.  

Par défaut, le programme affiche le cuirassé **Alsace**.<br> 
Des options permettent de changer le navire affiché (par ex. un petit bateau '-l' ou le cuirassé **Wisconsin** '-W').

<img src="./demo.gif" alt="Demo" width="50%"/>

## Utilisation

```bash
ws [options]
```
